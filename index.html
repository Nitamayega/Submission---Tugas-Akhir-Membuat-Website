Proyek

Aplikasi Android Movie App dengan React Native

Movie App adalah aplikasi Android yang dirancang untuk menampilkan daftar film yang diperoleh dari API, termasuk fitur pencarian berdasarkan kategori dan kata kunci, serta penyimpanan film favorit. Aplikasi ini dikembangkan secara kolaboratif dalam tim sebagai ketua penggerak.

  • Mengembangkan fitur aplikasi Android menggunakan React Native, dengan fokus pada integrasi API untuk akses film, termasuk pencarian dan penyimpanan film favorit.
  • Mendesain antarmuka pengguna yang intuitif dan menarik, dengan memanfaatkan secara maksimal endpoint dari API.
  • Mendeploy aplikasi agar dapat digunakan oleh pengguna lain tanpa melibatkan rangkaian kode dengan penggunaan Expo Go.

Aplikasi Android Hydros (Hydroponics Autonomous)

Hydros adalah aplikasi khusus untuk mendeteksi penyakit pada tanaman hidroponik, termasuk sayur-sayuran dan buah-buahan. Aplikasi ini berfokus pada pemantauan kesehatan tanaman hidroponik yang bertujuan untuk meminimalkan kerugian hasil panen dengan deteksi dini penyakit tanaman. Peran pada proyek ini adalah Mobile Developer satu-satunya dalam tim.

  • Mengembangkan fitur aplikasi Android dengan Kotlin, fokus pada deteksi penyakit pada tanaman hidroponik untuk meminimalkan kerugian hasil panen.
  • Mengintegrasikan layanan Firebase ke dalam aplikasi Android, termasuk Firebase Authentication untuk keamanan pengguna dan Firestore untuk manajemen database real-time.
  • Mendesain antarmuka pengguna yang ramah dan menarik dengan menggunakan pedoman Material Design dari Android.
